Resumo Executivo
Visao Geral do Projeto
O monkai-trace e um SDK Python oficial para monitoramento e observabilidade de agentes de IA. Oferece clientes sincrono e assincrono, integracoes com LangChain, OpenAI Agents, gerenciamento de sessoes, batch processing e exportacao de dados. A auditoria avaliou 12 modulos, 20+ exemplos e 10+ documentos.
Sync Client
HTTP client com retry, batch upload e session mgmt
Async Client
aiohttp com uploads paralelos via asyncio.gather
Pydantic Models
Validacao tipada de records, logs, tokens e transfers
Integracoes
LangChain, OpenAI Agents, MonkAI Agent, Python logging
